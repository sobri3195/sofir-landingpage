# SOFIR Landing Page - Legacy HTML Version

> ⚠️ **Note**: This is the legacy pure HTML/CSS/JS version. The main version is now React-based.
> This file is kept for backward compatibility and reference.

## About This Version

This is the original static HTML version of SOFIR landing page:
- Pure HTML5, CSS3, Vanilla JavaScript (ES5)
- No build process required
- Can be opened directly in browser
- Lightweight and simple

## Files

- `index-legacy.html` - Main HTML file (legacy version)
- `css/styles.css` - Stylesheet
- `js/main.js` - JavaScript functionality

## Quick Start

```bash
# Option 1: Open directly
open index-legacy.html

# Option 2: Local server
python -m http.server 8000
# Then open http://localhost:8000/index-legacy.html
```

## Deployment

Upload these files to any static hosting:
- `index-legacy.html`
- `css/` folder
- `js/` folder
- `favicon.svg`
- `manifest.json`

## Migration to React

The project has been migrated to React for better maintainability and developer experience.

**To use React version:**
1. See main `README.md`
2. Run `npm install`
3. Run `npm run dev`

## Features

All features from the React version, but implemented in vanilla JavaScript:
- Sticky header
- Mobile navigation
- Animated counters
- FAQ accordion
- Smooth scrolling
- Lazy loading
- Intersection Observer animations
